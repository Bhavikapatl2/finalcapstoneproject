Exploratory Data Analysis (EDA) was performed on the Paisabazaar banking fraud dataset to understand the structure and patterns in the data. The dataset was examined for data types, missing values, and key financial variables. Visualizations such as bar plots were used to explore relationships between variables like credit score and total EMI per month. The analysis helped identify trends in customer financial behavior and potential risk factors. Overall, EDA provided insights that can support further fraud detection and risk analysis.
